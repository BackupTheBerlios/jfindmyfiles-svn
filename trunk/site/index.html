News

JfindMyFiles is released!

After about 4 months of development the first official release is here. Not much of a release I must admit, but with the end of my course's project, and my graduation, I plan to improve the existing code and work hard on the software so that all objectives are achieved.

The current release targets developers and not end users as the software is not yet usable, though much of what were the objectives set forth in the beginning are implemented, there are lots of usability and stability issues to consider this software usable.

Along with the release comes the new site, the HTML pages that were here before had no information, just a bunch or ranting on what I wanted to create.

Now I'm going to take a brake from all my projects, JFindMyFiles included, and after a period in which I intend to recharge my batteries, I'll come back to update the software to a new platform, correct many of the existing bugs and release the 1.0 version. See you soon.
