JFindMyFiles Feature List

The listed features are those I intend to provide in the software and not those already available, as stated in the home page, I'm having a break from the development, until then, you can at leat see what the software will have.

  • Platform independent software allowing you to use any operation system for which a Java is available;
  • Can catalog you hard drives, CD-ROM drives, floppies, mapped and normal network folders;
  • Allows for simple search or using regular expressions;
  • Allows exporting of data to XML and CSV formats and export using HTML templates that you can create;
  • Using the Plugin API you can add new file types or provide information from different sources to known types.
