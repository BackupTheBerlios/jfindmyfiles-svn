Submitting a bug is something very important for developers. They are often in need of feedback on their projects. You can in BerliOS submit a bug for any project. For this, you just need to be logged in. Submitting a bug Just go on the Bug Tracking system for a project, and then click on Submit. First, you must choose a category and a group for the bug you will submit. Categories and groups are set up by a project administrator. Categories are equivalent to project parts (i.e : Server, Graphics, Engine , ...), and Bug groups define types of bugs (i.e : Unreproducible, Missing Feature ...). If no categories or bug groups are defined, just let "none" and a project administrator will classify the bug himself. After that, enter a quick description of the bug in the summary field. This description must be as clear as possible. Then enter the details in the details Box. And click on the Submit button to end this bug report. You'll be noticed when the bug will be closed.
